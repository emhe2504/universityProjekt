# UniversityProjekt

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.7.


Länk till webbplatsen: https://nyauniversitetet.onrender.com


Nya Universitet är ett fiktivt universitet som erbjuder utbildningar och kurser inom en bredd av ämnen. Webbplatsen består av tre delar:

____

Förstasidan som innehåller information om universitetet. Exempel då delar på startsidan är information om universitetet, kurser de erbjuder, när universitetets ansökningsperiod är 
samt ett kalendarium med viktiga händelser.

____

En undersida som presenterar universitetets alla kurser. Här finns kurserna presenterade i en tabell med möjlighet att sortera kurser och filtrera kurser utifrån
kurskod och kursnamn. Det går även att välja att endast kurser inom ett specifikt ämne ska visas. Användaren kan även lägga till kurser härifrån i ett ramschema.

____

Slutligen en undersida där användaren hittar sitt egenskapade ramschema. Även dessa presenteras i en tabell. Detta fungerar med hjälp av lagring i localStorage. 
Ävendaren kan även ta bort kurser från ramschemat.
